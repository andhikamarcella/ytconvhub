# YTConv Hub 2.1

Direktori resmi untuk seluruh layanan YTConv.

## Perubahan 2.1

- Tampilan Hub dirombak menjadi service directory yang lebih khas dan tidak memakai kartu template biasa.
- Pulse diarahkan ke `https://ytconvpulse.vercel.app/`.
- Tema Terang, Gelap, dan Otomatis mengikuti perangkat.
- Search bar mencari nama layanan, perangkat, fitur, dan masalah umum.
- Keyboard: tekan `/` atau `Ctrl/Cmd + K` untuk fokus pencarian.
- Filter layanan: Semua, Mulai, Belajar, Bantuan, dan Pantau.
- Tanpa framework dan tanpa proses build.

## Deploy

Upload seluruh isi folder ini ke GitHub, lalu impor repository ke Vercel.
